# 🌐 Artiordex – Digital & Automation Transformation Company

> AI · Cloud · Automation 기반 DX/AX 혁신 파트너

## 🚀 Artiordex 소개

**Artiordex(아티올덱스)** 는 **AI, Cloud, Automation 기술을 기반으로 기업의 Digital Transformation(DX)과 Automation Transformation(AX)** 을 실현하는 기술 중심 기업입니다.

우리는 단순한 소프트웨어 개발 회사를 넘어, 데이터 기반 의사결정, 자동화, 지능형 서비스, 그리고 인간 중심의 디지털 경험(UX)을 함께 만드는 **Engineering Partner**입니다.

### 핵심 철학

Artiordex는 다음의 핵심 철학을 중심으로 움직입니다.

- **Automation First** – 복잡함을 단순하게, 반복을 자동화로 대체
- **AI Native** – AI가 하나의 플러그인이 아닌, 시스템 기본 구조가 되는 설계
- **Cloud Everywhere** – 비용 효율성과 유연성을 극대화하는 클라우드 중심 운영
- **Experience Driven** – 기술보다 경험이 앞선다는 UX 중심 사고

우리는 이러한 철학을 바탕으로 교육, 제조, 디지털 서비스, 스타트업 제품 개발까지 다양한 영역에 기술을 적용하고 있습니다.

## 🏗️ Artiordex 핵심 제품·솔루션 (Arti Suite)

Artiordex는 다음과 같은 자체 솔루션들을 개발·고도화하고 있습니다.

### 🔸 ArtiFlow – DX 자동화 플랫폼
복잡한 워크플로우를 자동화하고, API 기반 통합을 통해 기업의 "일하는 방식"을 근본적으로 바꾸는 플랫폼입니다.

### 🔸 ArtiSync – 클라우드 기반 통합 데이터 허브
분산된 데이터를 하나로 연결하고 AI 기반 정제·처리·추론형 데이터 파이프라인을 구성합니다.

### 🔸 ArtiQA – Playwright 기반 테스트 자동화 솔루션
UI 자동화, 성능 테스트, 회귀 테스트를 자동화하여 개발·운영 비용을 크게 절감하는 QA 자동화 툴킷입니다.

### 🔸 ArtiLabs – 교육 & R&D AI 플랫폼
학습·교육·연구 분야에 특화된 AI 기반 콘텐츠/평가/자동화 솔루션.

## 🧩 Artiordex가 만드는 경험

우리가 만드는 제품과 서비스는 단순 "기능"이 아니라 **전문가 경험(Expert Experience, EX)** 을 담고 있습니다.

- 사용자를 기다리게 하지 않는 빠른 UI
- AI 기반 예측형 흐름으로 사용자의 의도를 먼저 파악
- 모바일·데스크톱·태블릿 어디서든 균형 잡힌 UX
- 복잡한 데이터를 한눈에 파악하는 시각화
- 확장 가능한 구조로 MVP → Enterprise까지 자연스러운 스케일업

> Artiordex는 기술이 보이지 않을수록 좋은 기술이라고 믿습니다.
> 사용자는 화면만 보고, 시스템은 알아서 작동하는 구조.
> 그게 우리가 만드는 방향입니다.

## ⚙️ Engineering Philosophy

Artiordex의 개발 문화는 다음 네 가지 원칙으로 요약됩니다.

### 1) Simplicity First
복잡성은 기술의 적입니다. 불필요한 레이어, 중첩된 의존성, 무거운 인프라를 피하고 가능한 가장 단순한 형태를 찾습니다.

### 2) Modular & Layered Architecture
컴포넌트화된 UI, 도메인 레이어, 데이터 어댑터 구조 등 유지보수와 확장을 전제로 설계합니다.

### 3) AI-Driven Development
기능 하나를 개발해도 "AI를 얹을 수 있을까?"를 먼저 고민합니다. AI는 선택이 아니라 기본 기능입니다.

### 4) Performance as a Feature
성능은 품질의 일부이며, 빠른 로딩·경량 API·최적화된 쿼리·효율적 렌더링을 통해 사용자 경험을 완성합니다.
